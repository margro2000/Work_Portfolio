Building A Design Thinking Collab Site Using React

I'm a strong believer in using Design Thinking to make amazing, easy to use and intuitive products that make people's lives better. Sometimes it's hard to implement these principles for teams. Using The Graph to Visualize Ethereum Data

The Graph was an awesome tool to be able to visualize the history of Ethereum usage and track interesting metadata

The History of Etheruem Data Set This table provides an indexed history of each ethereum block starting from block 49880. It uses the subgraph Ethereum Blocks which uses TerraNullius contract as the source. This project utilizes The Graph to collect all data, connects to my coinbase wallet to generate a key, and send the data to the browser with React.
